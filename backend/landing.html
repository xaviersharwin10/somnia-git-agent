1 Install CLI

git config --global alias.somnia-agent '!git-somnia-agent'
2 Initialize Repository
cd your-agent-repo
git somnia-agent init
git somnia-agent secrets set GROQ_API_KEY=your-key
3 Authorize GitHub (Above) or Manual Webhook
Manual: GitHub ‚Üí Settings ‚Üí Webhooks ‚Üí Add webhook
URL: https://somnia-git-agent.onrender.com/webhook/github/push
Events: Just the push event
4 Deploy
git push origin main

This automatically deploys an Agent.sol contract on Somnia and starts your agent!
