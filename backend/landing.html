üìã Quick Start Guide

1 Install CLI
npm install -g git-somnia-agent
git config --global alias.somnia-agent '!git-somnia-agent'
2 Initialize Repository
cd your-agent-repo
git somnia-agent init
git somnia-agent secrets set GROQ_API_KEY=your-key
git somnia-agent secrets set AGENT_PRIVATE_KEY=0x-your-key
3 Authorize GitHub (Above) or Manual Webhook
Manual: GitHub ‚Üí Settings ‚Üí Webhooks ‚Üí Add webhook
URL: https://somnia-git-agent.onrender.com/webhook/github/push
Events: Just the push event
4 Deploy
git push origin main

This automatically deploys an Agent.sol contract on Somnia and starts your agent!

üîÑ A/B Testing with Git Branches

Each Git branch becomes a separate agent contract on Somnia. Test multiple strategies in parallel!

1 Create Strategy Branch
git checkout -b aggressive-strategy
# Modify agent.ts with your strategy
git add agent.ts
git commit -m "Aggressive trading strategy"
2 Deploy Parallel Agent
git push origin aggressive-strategy

Now you have 2 agents running simultaneously, each with its own smart contract!

3 Compare Performance
git somnia-agent compare main aggressive-strategy

View side-by-side performance metrics for all your strategies.
